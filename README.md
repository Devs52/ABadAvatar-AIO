🚀 Quick Start
Download the AIO package

Drag and drop the contents to your USB drive

Put the USB drive into your xbox 360 console

That's it. Everything is pre-configured and ready to go.

🎛️ Changing the Dashboard
Edit launch.ini and change line 37 to point to your preferred dashboard.

Example:

ini
Default = Usb:\Metro2\dash.xex   # Metro V2 (default)
Default = Usb:\NXE\dash.xex      # NXE
